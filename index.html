About

This project is to develop a web-based calculator that can process the four basic functions, as well as complex bracket structures.


The outputs of the calculator were tested against 35 different test cases that were written beforehand, to ensure all operations functioned as they should. The details of the test cases can be found in the README in the GitHub Repo.


Use your mouse to click on the calculator buttons, and see it in action!
